# System Widgets

A library of pre-built, theme-aware widgets that wrap Flutter primitives and enforce the project's design system. Use these instead of raw Flutter widgets to ensure consistent styling across the app.

## Directory Structure

```
system/
├── app_button.dart           # Primary / secondary / text buttons
├── app_chip_widget.dart      # Filter / selection chips
├── app_divider.dart          # Styled horizontal divider
├── app_selection_box.dart    # Checkbox / radio-style selection box
├── app_swtich.dart           # Toggle switch
├── default_app_bar.dart      # Pre-styled AppBar
├── icon/
│   └── app_icon_widget.dart  # Icon + theme size/colour resolver
├── images/
│   ├── app_cached_network_image.dart  # CachedNetworkImage wrapper
│   ├── app_image_widget.dart          # Asset / network / file image router
│   └── app_svg_picture_widget.dart    # SVG renderer
├── loaders/
│   ├── app_circular_loader.dart       # Circular progress indicator
│   └── app_dialog_loader.dart         # Full-screen blocking loader dialog
└── text/
│   └── app_text_widget.dart           # Text with named styles from theme
└── text_fields/
    ├── app_dropdown.dart              # Dropdown selector
    ├── app_text_filed.dart            # Standard text input
    └── base/
        └── custom_text_form_field_widget.dart  # Low-level base field
```

## `AppButton`

Themed button that maps to your `AppButtonTheme`:

```dart
AppButton.primary(
  label: 'Save',
  onPressed: _save,
  isLoading: state.isLoading,
)

AppButton.secondary(
  label: 'Cancel',
  onPressed: () => context.router.pop(),
)

AppButton.text(
  label: 'Skip',
  onPressed: _skip,
)
```

---

## `AppTextWidget`

Uses named text styles from the theme. Never set raw `TextStyle` — use this:

```dart
AppTextWidget(
  text: 'Hello',
  style: context.textStyles.titleLarge,
  maxLines: 2,
  overflow: TextOverflow.ellipsis,
)
```

## `AppTextField`

Styled text input backed by `CustomTextFormFieldWidget`:

```dart
AppTextField(
  label: 'Email',
  hint: 'you@example.com',
  controller: _emailController,
  keyboardType: TextInputType.emailAddress,
  validator: context.emailValidator(),
  prefixIcon: const Icon(Icons.email),
)
```

## `AppDropdown`

```dart
AppDropdown<String>(
  label: 'Country',
  items: countries,
  value: _selected,
  itemLabel: (c) => c.name,
  onChanged: (c) => setState(() => _selected = c),
)
```

## `AppImageWidget`

Routes to the correct image source automatically based on `AppImageType`:

```dart
// Asset
AppImageWidget(
  type: AppImageType.asset,
  path: Assets.images.logo,
)

// Network (with caching)
AppImageWidget(
  type: AppImageType.network,
  path: 'https://example.com/avatar.jpg',
  width: 80, height: 80,
  borderRadius: BorderRadius.circular(40),
)

// File
AppImageWidget(
  type: AppImageType.file,
  path: file.path,
)
```

## `AppSwitch`

```dart
AppSwitch(
  value: isDarkMode,
  onChanged: (v) => context.read<AppSettingBloc>().add(
    UpdateSettingEvent(data: (s) => s.copyWith(isDarkMode: v)),
  ),
)
```

## Copying to a New Project

1. Copy the entire `system/` folder.
2. Ensure `cached_network_image`, `flutter_svg`, `flutter_screenutil` are in `pubspec.yaml`.
3. The widgets reference `context.colors`, `context.textStyles`, and `context.appTheme` — copy the [theme system](../../../../../config/themes/README.md) as well.
4. Update `AppImageType` enum if you need additional image source types.
