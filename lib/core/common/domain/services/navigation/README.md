# Navigation Service

A singleton facade over [`auto_route`](https://pub.dev/packages/auto_route)'s `AppRouter` that provides a single, injectable entry point for programmatic navigation anywhere in the app — including from services and BLoCs that have no `BuildContext`.

## Files

| File | Purpose |
|------|---------|
| `navigation_service.dart` | Singleton wrapper around `AppRouter` |
| `../../../presentation/routes/app_router.dart` | Route declarations (generated by `auto_route`) |

## Dependencies

```yaml
dependencies:
  auto_route: ^11.1.0
  injectable: ^2.x.x
  get_it: ^9.x.x

dev_dependencies:
  auto_route_generator: ^10.x.x
  build_runner: ^2.x.x
```

## Setup

`NavigationService` is decorated with `@injectable` and receives `AppRouter` via constructor injection. `AppRouter` itself is registered as `@Singleton()`.

After running the build runner, both are available via `GetIt`:

```bash
dart run build_runner build --delete-conflicting-outputs
```

## Usage

### Resolve from GetIt (services / BLoCs)

```dart
final nav = sl<NavigationService>();

// Push a new route
await nav.push(const HomeRoute());

// Replace current route
await nav.replace(const LoginRoute());

// Replace with a named path (deep link / notification tap)
await nav.replaceNamed('/home/profile');

// Pop
nav.popForced();
await nav.maybePop();     // safe pop — returns false if nothing to pop
nav.popUntilRoot();

// Push an inline widget (no route declaration needed)
final result = await nav.pushWidget<bool>(
  context: context,
  widget: const ConfirmationDialog(),
);

// Replace entire stack (e.g., after logout)
await nav.replaceAll([const LoginRoute()]);

// Close a specific named route (pops if on top, removes otherwise)
await nav.closeSpecificRoute(name: ProfileRoute.name);
```

### Check current route

```dart
final current = sl<NavigationService>().current;
print(current.name);   // e.g., 'HomeRoute'

final isAlreadyOpen = nav.checkIfAlreadyOpened(name: HomeRoute.name);
```

## Adding Routes

1. Create a route file with `@RoutePage()`:

```dart
// lib/features/home/presentation/routes/home_route.dart
import 'package:auto_route/auto_route.dart';

@RoutePage()
class HomeScreen extends StatelessWidget { ... }

class HomeRoute {
  static const AutoRoute route = AutoRoute(page: HomeRoute.page);
}
```

2. Register in `AppRouter`:

```dart
// lib/core/common/presentation/routes/app_router.dart
@override
List<AutoRoute> get routes => [
  HomeRoute.route,
  SplashRoute.routes,
  PermissionRoute.routes,
  ..._OverlayRoutes.all,
];
```

3. Regenerate:

```bash
dart run build_runner build --delete-conflicting-outputs
```

## Copying to a New Project

1. Copy `navigation_service.dart`.
2. Copy `app_router.dart` and `app_router.gr.dart` (generated).
3. Add `auto_route` and `auto_route_generator` to `pubspec.yaml`.
4. Ensure `AppRouter` is annotated with `@Singleton()` and `@AutoRouterConfig()`.
5. Wire `AppRouter` into `MaterialApp.router` inside `App.build()`.
6. Run `build_runner`.
